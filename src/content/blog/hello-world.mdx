## コードブロックのテスト

Python のサンプル:

```python
def fibonacci(n: int) -> int:
    """フィボナッチ数列のn番目を返す"""
    if n <= 1:
        return n
    a, b = 0, 1
    for _ in range(2, n + 1):
        a, b = b, a + b
    return b

print(fibonacci(10))  # 55
```

TypeScript のサンプル:

```typescript
interface Post {
  title: string;
  date: Date;

}

const formatDate = (date: Date): string => date.toISOString().slice(0, 10);

  title: "はじめての記事",
  date: new Date("2026-03-31"),
  tags: ["test", "math"],

console.log(formatDate(latestPost.date)); // "2026-03-31"
```
